# Machine Learning & Deep Learning Project

Course project for Machine Learning & Deep Learning  
School of Artificial Intelligence and Data Science  
Academic Year: 2024–2025

## Project Overview

This project contains practical implementations and experiments related to machine learning and deep learning topics covered during the course.

The work includes:

- Classical classification algorithms
- Model evaluation and comparison
- Dimensionality reduction techniques
- Deep learning fundamentals
- CNN experiments
- Visualization and analysis of results

The project was implemented using Python and Jupyter Notebooks.
